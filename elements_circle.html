Circle¶

Represent a circle element of an SVG.

SVG Documentation

class Draw.Circle(x=0, y=0, r=0)¶
__init__(x=0, y=0, r=0)¶
Parameters:

  • x – x coordinate of the center of the circle.

  • y – y coordinate of the center of the circle.

  • r – radius of the circle.

All location/size parameters Can be pixels, percent, or any of the supported units (see Units Documentation for details on what units are supported). Default is pixel values.

property fill¶

Get or Set a tuple of r,g,b values for the fill of this element.

property fill_opacity¶

Get or Set Number Value from 0 to 1 that controls the opacity of the fill. 0 -> Totally Transparent 1 -> Totally Opaque

property gradient¶

Get or Set name of the gradient to define the fill with. Gradient will reference a Gradient.Linear object. Gradient.Linear objects need to be added to the defs field of the SVG.SVG object.

property r¶

radius of circle.

property stroke¶

Get or Set a tuple of r,g,b values for the stroke/outline of this element. :

property stroke_dasharray¶

Get or Set Iterable of integers that defines the pattern of dashes to use. Each element in the iterable defines the width in pixels for the next dash element in the pattern.

property stroke_opacity¶

Get or Set Number Value from 0 to 1 that controls the opacity of the stroke/outline. 0 -> Totally Transparent 1 -> Totally Opaque

property stroke_width¶

Get or Set width of the stroke/outline of this element. Numeric Values.
