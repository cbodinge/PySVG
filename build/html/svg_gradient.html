Gradients¶

Build a gradient definition to apply to element fills, strokes, etc.

To learn about SVG gradients see SVG Documentation

class Gradient.Linear(name: str)¶

Creates a Linear Gradient Object. The Gradient is defined in the defs section of the SVG and referenced by name in the parameters of the desired elements

__init__(name: str)¶
Parameters:

name – a string identifying this gradient. The name is how gradients are referenced in the target elements

add_stop(percent: float, color: tuple[int, int, int], opacity: float = 1.0)¶

Adds a stop to define gradient behavior. Stops are isstances of the Gradient.Stop class. This function appends the new Stop instance reference to the Linear._stops list.

Parameters:

  • percent (float) – Dictates where the stop is as a percentage of the total distance that the gradient will cover.

  • color (tuple) – RGB values defining the color at this stop.

  • opacity (float) – Decimal value defining the opacity at this stop.

construct()¶

Builds the gradient definition that will be instituted in the defs section of the parent SVG.

Returns:

String representation of this gradient object.

Return type:

str
